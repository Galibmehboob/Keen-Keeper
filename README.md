<h2>📝 Description</h2>
<p style="background:#ecfeff; padding:12px; border-radius:8px;">
  KeenKeeper is a simple and responsive web application that helps users manage and maintain their friendships. 
  It allows users to track interactions like calls, texts, and video chats, ensuring they stay connected with friends regularly.
</p>
